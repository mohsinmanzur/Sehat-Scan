# SehatScan Frontend Prototype

React Native + Expo TypeScript prototype for the SehatScan patient app.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   # or
   yarn
   ```

2. Run the app:

   ```bash
   npx expo start
   ```

3. Open on an emulator or Expo Go on your phone.

This project is **frontend-only** with mock data and no backend.
